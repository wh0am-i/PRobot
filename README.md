# PRobot
<h3>Code repository of a simple battle bot for balloons battle, which each robot carries a balloon on its back and a spike at the front. The project uses an Esp-32, H-Bridge, two dc motors and a couple of batteries.
<br>
<br>
This bot won the 1st place in the competition due to some mods at the gear ratio of the motors, as well as improvements at spike and code.</h3>
